## 普通函数跟箭头函数区别

|          | 语法格式                       | new 和原型 | argument super new.target | this 指向                                  | call.apply 和 bind |
| -------- | ------------------------------ | ---------- | ------------------------- | ------------------------------------------ | ------------------ |
| 普通函数 | function(){}函数声明函数表达式 | 有         | 有                        | 动态                                       | 修改 this 值       |
| 箭头函数 | ()=>{}函数表达式               | 没有       | 没有,可以调用外围         | 一般是全局对象或者被普通函数包含指向上一层 | 不可修改 this 值   |
